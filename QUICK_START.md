# KEIBA NAVIGATOR — 30分クイックスタート

このアプリを **本気で動かす** までの最短手順。非エンジニアでも進められるように、コピペ可能なコマンド形式で書いています。

## このアプリは何ができる?

- ✅ 出馬表・オッズの取得（要 JRA-VAN 契約）
- ✅ 期待値（EV）でレース全頭を採点
- ✅ 「買わないAI」 — 期待値マイナスのレースは見送りを推奨
- ✅ 自分の馬券記録から AI が育つ（★1〜★5）
- ✅ クラウド同期（メールでログインすれば PC・スマホで共有）
- ❌ **100% 当たる魔法はありません**（誰にもできない）。狙うのは「長期で回収率100%超」

---

## ステップ 0 : 公開アプリをまず触る (0分・無料)

スマホやPCのブラウザで **https://keiba-navigator.vercel.app** を開く。

- 設定 → クラウド同期 → メールアドレスを入れて「メールでログイン」
- 届いたメールのリンクをクリック → 同期完了

この時点では「天気」「ニュース」と「育成レベル UI」だけ動きます。実データを取るには次のステップへ。

## ステップ 2 : JV-Link をインストール (5分・Windows のみ)

JV-Link は JRA-VAN がオッズや出馬表をプログラムから取れるようにする部品。**Windows 専用**（Mac・スマホでは動きません）。

1. https://jra-van.jp/dlb/sdv/sdk.html を開く
2. **「JV-Link インストール」** リンクからダウンロード
3. ダウンロードした `.exe` をダブルクリック → 「次へ」「次へ」「インストール」（管理者パスワード要）
4. 完了したら、スタートメニューから **JV-Link** を起動
5. ステップ1でメモした **利用キー** を入れる

---

## ステップ 3 : 32bit Python を入れる (5分・無料)

JV-Link は古い Windows 部品（32bit）なので、**32bit 版の Python** が必要です。普通の Python（64bit）では動きません。

1. https://www.python.org/downloads/windows/ を開く
2. **「Windows installer (32-bit)」** を選んでダウンロード
3. ダブルクリックでインストール
4. ⚠️ インストール画面で **「Add Python to PATH」にチェック** を入れること
5. インストール完了後、**PowerShell** を開いて以下をコピペ実行：

```powershell
py -3.12-32 -m pip install pywin32
```

---

## ステップ 4 : このアプリをダウンロード (5分)

1. PowerShell を開く（Windows キー → 「powershell」と入力）
2. 以下をコピペ実行：

```powershell
cd $env:USERPROFILE
git clone https://github.com/shougihajime-eng/keiba-navigator.git keiba
cd keiba
npm install
```

> 💡 `git` が無い場合は https://git-scm.com/download/win から「Standalone Installer 64-bit」を入れてから再実行。

---

## ステップ 5 : 接続テスト (1分)

PowerShell で `keiba` フォルダにいる状態で：

```powershell
py -3.12-32 jv_bridge\jv_fetch.py init
```

- ✅ **`[OK] JVInit 成功`** と出れば接続完了
- ❌ エラーが出たら、最後の行をスクショして検索 or サポートへ

---

## ステップ 6 : ダッシュボードを起動 (即時)

```powershell
npm start
```

ブラウザで http://127.0.0.1:8765 を開く。「更新」ボタンを押すとデータ取得開始。

> 💡 PC のホーム画面ショートカットにしたい場合は、`start.bat` をダブルクリック。

---

## よくあるトラブル

| 症状 | 原因 | 対処 |
| --- | --- | --- |
| `'py' は内部コマンド〜` | Python が PATH に追加されていない | 再インストールで「Add Python to PATH」にチェック |
| `pywin32 not found` | 64bit Python が混在 | `py -3.12-32 -m pip install pywin32` で必ず 32bit 指定 |
| `[FAIL] JVInit -202` | 利用キー未設定 | JV-Link を再起動してキーを入れ直す |
| Vercel でデータが見えない | JV-Link は Windows ローカル専用 | ローカル PC で取得 → `git push` すると Vercel にも反映 |

## 毎日の使い方

1. 朝、PowerShell で `cd ~\keiba` → `npm start`
2. ブラウザで `http://127.0.0.1:8765`
3. 「更新」ボタンを押す（押した時だけデータ取得 → API 制限を避けるため）
4. 結論を見る → 「狙える」が出たら少額で記録 → 結果待ち
5. 確定したら、AI の育成レベル（★）が伸びる

**毎週・毎月の振り返り**：「記録」タブで回収率を見る。**3ヶ月続けて回収率100%超**を目指す。
