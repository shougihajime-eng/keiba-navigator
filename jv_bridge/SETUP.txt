================================================================
 競馬ダッシュボード セットアップガイド（非エンジニア向け・完全版）
================================================================

このアプリは「天気」と「ニュース」は契約なしで自動取得できます。
オッズ・出馬表・馬体重・血統・過去成績など本格的な競馬データを
自動取得するには、以下の3点が必要です。

★重要★ お金を払う前に「動くこと」を確認できます。
  JRA-VAN は 2 段階契約です:
    ステップA: 開発者登録 → 無料（仕様書とサンプルデータがもらえる）
    ステップB: データサービス契約 → 月額2,090円（実データ配信が始まる）
  ステップA だけで全テストを緑にできる設計です。
  万一動かなければ B（有料）を契約しなければ1円もかかりません。

----------------------------------------------------------------
[A-1] 開発者登録（無料）
----------------------------------------------------------------
1. ブラウザで https://developer.jra-van.jp/ を開く
2. 右上の「新規登録」または「会員登録」をクリック
3. メールアドレス・パスワード・氏名を入力して登録
   ※ クレジットカード入力は出ません（無料登録）
4. 登録メールから本登録を完了
5. ログイン → 「ダウンロード」メニューを開く

----------------------------------------------------------------
[A-2] SDK と仕様書をダウンロード
----------------------------------------------------------------
1. https://jra-van.jp/dlb/sdv/sdk.html を開く
2. 「JV-Link Ver4.x SDK」をダウンロード（zip ファイル）
3. zip を展開（C:\jvlink_sdk\ など分かりやすい場所に）
4. 展開先に以下のファイルがあるはず:
   - JVData仕様書.pdf  ← ★これが一番重要★
   - SampleData/*.bin  ← ★テスト用サンプルデータ★
   - JVLink_v4_xx.exe  ← JV-Link 本体インストーラ
5. JV-Link 本体は B 契約後にインストール（A だけなら不要）

----------------------------------------------------------------
[A-3] 32bit Python と必要パッケージを入れる（一度だけ）
----------------------------------------------------------------
JV-Link は 32bit COM コンポーネントなので、64bit Python では動きません。
必ず「32bit版」を入れてください。

A) Python 3.12 (32bit) をダウンロード
   ブラウザで以下のリンクを開いて、自動でダウンロードされる .exe を実行：
   https://www.python.org/ftp/python/3.12.4/python-3.12.4.exe
   ※ ファイル名に "amd64" が含まれていないことを確認 (含まれていたら 64bit)

B) インストーラを起動 → 必ず以下にチェック
   ☑ Add python.exe to PATH   (これを忘れると後で必ず詰みます)
   ☑ Install launcher for all users (おすすめ・任意)
   → 「Install Now」をクリック

C) インストール終了後、PowerShell を**新規に**開いて (重要・既存窓は古いPATH)
   下のコマンドを 1 行ずつコピペして実行:

   py --version                       (3.12.x と出れば OK)
   py -3.12-32 -m pip install --upgrade pip
   py -3.12-32 -m pip install pywin32 pytest

D) 動作確認（仕様書転記前でも全部 PASS or SKIP になる）:
   cd C:\Users\shoug\競馬
   py -3.12-32 -m pytest jv_bridge\tests -q

   → 「N passed, M skipped」と出れば OK。failed が無ければ成功。

【困った時の確認コマンド】
   py -0                              (インストール済 Python の一覧)
   py -3.12-32 -c "import sys; print(sys.maxsize <= 2**32)"
   → True と出れば 32bit。False なら 64bit を間違って入れている。

----------------------------------------------------------------
[A-4] 仕様書からフィールド配置を転記（30 分〜1 時間）
----------------------------------------------------------------
JVData仕様書.pdf を開いて、jv_bridge\jvdata_struct.py の TODO を埋めます。
これは Claude に手伝わせれば一気に進みます。仕様書の該当ページを開いて
スクリーンショット → Claude に「この表を jvdata_struct.py の RA に転記して」
と頼めば OK です。

転記すべきもの:
  - RA レコード (レース情報) の各フィールド offset / length
  - SE レコード (馬毎レース情報) の各フィールド offset / length
  - HR レコード (払戻) の各券種 offset と HR_PAYOUT_LAYOUT の null 部分
  - O1 レコードのヘッダ部分 offset

転記完了したら RECORD_COMPLETED["RA"]=True 等に変更し、再度
py -3.12-32 -m pytest jv_bridge\tests -q
を実行。fixtures に SDK の SampleData を置いておけば、より多くのテストが
緑になります。

----------------------------------------------------------------
[A-5] サンプルバイナリを fixtures に配置
----------------------------------------------------------------
SDK 同梱の SampleData/ から、レコード種別ごとに 1 つずつコピー:

  jv_bridge\fixtures\RA\sample_RA.bin
  jv_bridge\fixtures\SE\sample_SE.bin
  jv_bridge\fixtures\O1\sample_O1.bin
  jv_bridge\fixtures\HR\sample_HR.bin

配置したら再度:
  py -3.12-32 -m pytest jv_bridge\tests -q

★ここまで全部 passed になれば「あとは月額契約するだけ」状態★
★1 円もかかっていません★

================================================================
 ▼ ここから先は実際に競馬データを取得する段階（月額が発生する）
================================================================

----------------------------------------------------------------
[B-1] JRA-VAN Data Lab. を契約する（月額 2,090円・税込）
----------------------------------------------------------------
1. ブラウザで https://jra-van.jp/dlb/ を開く
2. 「お申込み」から会員登録
3. 利用キー（または会員ID）を発行
   ※無料お試しはありません。

----------------------------------------------------------------
[B-2] JV-Link 本体をインストール（Windows 専用）
----------------------------------------------------------------
1. A-2 でダウンロードした JVLink_v4_xx.exe をダブルクリック
2. 管理者権限でインストール
3. 起動して、B-1 で取得した利用キーを設定

----------------------------------------------------------------
[B-3] 接続テスト
----------------------------------------------------------------
PowerShell でこのフォルダ（競馬\）に移動して：

    py -3.12-32 jv_bridge\jv_fetch.py init

「[OK] JVInit 成功」と出れば接続できています。
エラーが出たときは jv_bridge\jv_fetch.py status で詳細を確認できます。

----------------------------------------------------------------
[B-4] 過去レースを一括取得（10年分）して features.json を作る
----------------------------------------------------------------
    py -3.12-32 jv_bridge\jv_fetch.py aggregate --dataspec RACE --fromtime 20140101000000
    py -3.12-32 jv_bridge\aggregate_features.py

これで data\jv_cache\features.json が生成され、騎手・コース別勝率が
AI 推定に効くようになります。

----------------------------------------------------------------
[B-5] レースデータを取得する（例：単複オッズ）
----------------------------------------------------------------
JRA レースID は 18桁:
    年(4) + 月(2) + 日(2) + 場コード(2) + 開催回(2) + 開催日(2) + R(2)

例: 2026年5月3日 東京11R → 2026050305020111

    py -3.12-32 jv_bridge\jv_fetch.py rt --dataspec 0B31 --raceid 2026050305020111

DataSpec の主な値:
    0B11 = 馬体重
    0B12 = 払戻
    0B14 = 単勝・複勝・枠連
    0B15 = 馬連
    0B16 = ワイド
    0B17 = 馬単
    0B20 = 3連複
    0B30 = 3連単
    0B31 = 単勝・複勝（速報）
    （正確な一覧は JV-Link インターフェース仕様書を参照）

----------------------------------------------------------------
[B-6] ダッシュボードを起動する
----------------------------------------------------------------
このフォルダで start.bat をダブルクリック。
ブラウザで http://127.0.0.1:8765 を開く。

ダッシュボードの「データ取得状況」で
JV-Link が "ready" になっていれば、取得済みデータが表示されます。
